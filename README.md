webglchip8
==========
Still things todo!

Camera controls
---------------
Use mousedrags to move camera, mousewheel to zoom, alt + mousedrag for panning and shift + mousedrag for roll

Game controls
-------------
Each game has different keyboard mappings and are mapped to a more inituative keys, so try arrow keys/wsda space/enter.

Good Luck!
